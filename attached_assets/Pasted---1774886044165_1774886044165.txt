사주앱의 홈 화면을 전면 재정리해주세요.

앱 이름은 “나의 흐름”입니다.

이 앱은 전통적인 점술앱이 아니라
“나와 상대의 흐름, 오늘의 에너지, 관계와 궁합을 차분하게 이해하는 모바일 인사이트 앱”입니다.

이번 작업은 홈 화면만 먼저 다듬는 것이 목표입니다.

-----------------------------------
[핵심 목표]
-----------------------------------

현재 홈 화면은 정보는 어느 정도 들어가 있지만
아직 아래 문제가 있습니다.

1. 캐릭터가 너무 작아서 브랜드 핵심 요소로 느껴지지 않음
2. 컬러 시스템이 애매해서 상태 정보(좋음/보통/주의)가 직관적으로 전달되지 않음
3. 오늘의 핵심 메시지가 다소 짧고 건조해서 감성적 몰입이 약함
4. 영역별 오늘의 운 카드가 공간을 많이 차지하는데 비해 내용 밀도가 낮음
5. 버튼 스타일과 카드 스타일이 통일되지 않음
6. 쉐도우가 있어서 토스처럼 정돈된 느낌이 약함
7. “오늘의 기운”과 “오늘의 한마디”가 분리되어 있어 메시지가 중복됨
8. 로그인 이후 사용자의 닉네임이 반영되는 개인화 구조가 필요함

홈 화면은
“오늘 나의 흐름을 가장 먼저 감각적으로 이해하는 화면”
처럼 보여야 합니다.

즉,
정보 나열형 홈이 아니라
브랜드 + 감정 + 행동 가이드가 함께 보이는 홈이어야 합니다.

-----------------------------------
[디자인 톤앤매너]
-----------------------------------

원하는 방향은
“토스처럼 깔끔한 정보 정리”
+
“감성적인 캐릭터 중심 히어로”
+
“너무 칙칙하지 않은 컬러감 있는 앱”

입니다.

현재처럼 너무 muted beige / 회색 / 밋밋한 톤은 피하고 싶습니다.

원하는 느낌:
- 깔끔함
- 따뜻함
- 생동감
- 감성적이지만 과하지 않음
- 정리된 정보 구조
- 브랜드 캐릭터가 살아있는 UI

피해야 할 것:
- 칙칙한 베이지 일변도
- 과한 그림자
- 전통 점술앱 느낌
- 카드 안에 카드가 많은 복잡한 구조
- 버튼 스타일이 제각각인 상태

-----------------------------------
[컬러 시스템 정의]
-----------------------------------

앱 전반에 상태 컬러 체계를 명확히 정의해주세요.

상태 컬러 기준 예시:
- 좋음 = 파랑 계열
- 보통 = 초록 계열
- 주의 = 주황 계열
- 나쁨/경고 = 빨강 계열

이 상태 컬러는
에너지 바, 배지, 상태 칩, 운세 강조 문구 등에 일관되게 적용해주세요.

중요:
현재처럼 모든 바와 수치가 비슷한 컬러로 보이지 않게 해주세요.
사용자가 한눈에 “오늘 어느 영역이 좋고, 어느 영역이 조심해야 하는지” 구분할 수 있어야 합니다.

기본 배경 톤은 따뜻한 아이보리/크림 계열로 유지하되,
상태 표현에는 확실한 컬러 차이를 주고 싶습니다.

-----------------------------------
[홈 화면 구조]
-----------------------------------

홈 화면은 아래 구조로 재정리해주세요.

1. 상단 앱바
- 왼쪽: 앱 이름 “나의 흐름”
- 오른쪽: 로그인 / 프로필 진입
- 로그인 후에는 사용자가 닉네임을 설정할 수 있게 하고,
  홈 화면 타이틀에도 그 닉네임이 반영되도록 설계해주세요.

예:
“소연님의 오늘 흐름”
“OO님의 오늘”

2. 히어로 카드 (가장 중요)
이 영역은 홈 화면의 핵심입니다.

포함 요소:
- 날짜
- 닉네임 기반 타이틀 (예: 소연님의 오늘 흐름)
- 오늘의 핵심 메시지
- 오늘 신경 써야 할 감성적 행동 가이드 문장
- 오늘의 키워드 칩 2~4개
- 큰 캐릭터 이미지

중요:
캐릭터는 지금보다 훨씬 크게 넣어주세요.
장식용 아이콘처럼 작게 들어가는 것이 아니라,
홈 화면의 감정 중심축이 되는 hero visual이어야 합니다.

캐릭터는 오른쪽 상단 작은 배치가 아니라
시각적으로 확실히 존재감 있게 들어가야 합니다.
오늘의 에너지/일진/키워드에 따라 캐릭터가 바뀌는 구조를 전제로,
“오늘의 가이드” 같은 느낌이 나야 합니다.

오늘의 핵심 메시지는 지금보다 더 감성적이고 실질적이어야 합니다.
단순 요약 한 줄보다,
“오늘 무엇을 의식하면 좋은지”가 느껴지는 문장으로 바꿔주세요.

예시 방향:
- 오늘은 급하게 결론 내리기보다 마음을 한 번 더 살피는 편이 좋습니다.
- 오늘은 내 감정보다 관계의 온도를 먼저 살펴보면 흐름이 부드러워집니다.
- 오늘은 나를 밀어붙이기보다 안정감을 지키는 선택이 더 잘 맞습니다.

즉,
“운세 설명”보다 “감성적인 행동 가이드”가 필요합니다.

3. 오늘의 에너지 섹션
현재 구조는 유지하되 더 정돈되게 바꿔주세요.

포함 항목 예시:
- 감정 안정
- 결정력
- 관계 흐름

개선 방향:
- 상태 점수에 따라 컬러가 달라지게 하기
- 좋음/보통/주의가 시각적으로 바로 읽히게 하기
- 바 디자인을 더 토스식으로 정리하기
- 라벨 / 수치 / 바 간 간격을 더 정리하기

쉐도우는 제거하고,
깔끔한 border와 spacing 중심으로 정리해주세요.

4. 오늘의 흐름 구조 섹션
현재의 일진 / 대운 / 세운 / 월운은 홈에서 보여주는 것이 좋습니다.
다만 지금처럼 조금 뜬금없이 보이지 않게
한 섹션 안에서 정리된 구조로 보여주세요.

예시:
- 오늘 일진
- 대운
- 세운
- 월운

이 정보는 홈에 있어도 되지만,
과하게 설명을 늘리지는 말고
한눈에 현재 흐름을 읽을 수 있는 compact summary card 구조로 보여주세요.

5. 영역별 오늘의 운 섹션
현재 가장 아쉬운 부분입니다.

문제:
카드가 차지하는 공간에 비해 내용이 너무 짧고 건조합니다.

개선 방향:
각 카드가 단순히
“관계운 / 보호·신뢰 관계”
처럼 라벨만 말하는 구조가 아니라

“오늘 실제로 어떻게 행동하면 좋은지”
를 짧지만 감성적으로 안내하는 카드가 되어야 합니다.

예시:
관계운
- 오늘은 먼저 반응하기보다 한 박자 듣는 태도가 관계를 편안하게 만듭니다.

건강운
- 무리해서 속도를 올리기보다 컨디션을 회복하는 흐름에 집중하는 편이 좋습니다.

일/학업운
- 배움과 정리에 집중하면 성과보다 더 중요한 방향성이 잡히는 날입니다.

재물운
- 큰 판단보다 지출 흐름을 점검하는 작은 정리가 더 도움이 됩니다.

즉,
“상태 라벨”만 쓰지 말고
짧은 감성 문장 + 행동 방향이 함께 들어가게 해주세요.

카드 수는 4개를 유지해도 되지만
내용 밀도를 높이고,
레이아웃은 지금보다 더 정리된 그리드 형태로 다듬어주세요.

-----------------------------------
[오늘의 한마디 처리]
-----------------------------------

하단의 “오늘의 한마디” 별도 카드는 삭제하거나 합쳐주세요.

이유:
지금은 히어로 메시지와 중복되는 느낌이 있습니다.

대신
오늘의 핵심 메시지 + 오늘의 행동 가이드 문장을
상단 hero 안에 더 잘 녹여주세요.

즉,
메시지는 분산하지 말고
상단 히어로에 집중시키고 싶습니다.

-----------------------------------
[버튼 / 카드 / UI 컴포넌트 규칙]
-----------------------------------

앱 전체에서 버튼 스타일을 통일해주세요.

현재처럼 버튼/칩/카드 스타일이 제각각이면 안 됩니다.

규칙:
- primary button
- secondary button
- text button

이 3종 정도로 통일해주세요.

카드 스타일:
- 라운드 일관성 통일
- 그림자 최소화 또는 제거
- border + background tone + spacing으로 구분
- 토스처럼 정돈된 카드 구조

쉐도우는 선호하지 않습니다.
그림자로 띄우기보다
여백과 테두리, 정렬로 깔끔하게 보이게 해주세요.

칩/배지 스타일도 통일해주세요.
키워드 칩과 상태 칩이 서로 다른 언어처럼 보이지 않게 해주세요.

-----------------------------------
[개인화]
-----------------------------------

로그인 후 닉네임 설정이 가능해야 합니다.

홈 화면의 문구도 닉네임 기반으로 자연스럽게 바뀌게 해주세요.

예:
- 소연님의 오늘 흐름
- 민지님의 오늘 흐름

즉, 홈은 사용자가 앱에 들어왔을 때
“내 화면 같다”는 느낌이 들도록 개인화되어야 합니다.

-----------------------------------
[브랜드 방향]
-----------------------------------

이 앱은 전통 사주앱보다
“나와 상대, 오늘의 흐름을 감성적으로 읽는 앱”
처럼 느껴져야 합니다.

따라서 홈 화면도
텍스트 나열이 아니라
브랜드 캐릭터 + 감성적 메시지 + 정돈된 인사이트 카드
구조가 되어야 합니다.

캐릭터는 매우 중요한 브랜드 자산이므로
홈에서는 반드시 크게, 존재감 있게 활용해주세요.

-----------------------------------
[정리]
-----------------------------------

이번 홈 화면 리디자인에서 가장 우선 적용할 것:

1. 앱 이름 “나의 흐름” 반영
2. 닉네임 기반 개인화 타이틀
3. 캐릭터 hero 영역 크게 확대
4. 오늘의 핵심 메시지를 더 감성적이고 실질적인 행동 가이드 문장으로 개선
5. 좋음/보통/주의/경고 상태 컬러 시스템 정의 및 적용
6. 오늘의 에너지 바 색상 상태별 구분
7. 영역별 오늘의 운 카드 내용 밀도 강화
8. 오늘의 한마디 카드 제거 또는 hero 메시지와 통합
9. 버튼 스타일 통일
10. 쉐도우 제거, 더 토스식으로 정돈된 카드 UI로 정리

결과물은
- 모바일 홈 화면 최종 UI 시안
- 필요한 경우 구조 개선안
- 컴포넌트 스타일 통일안
까지 포함해서 제안해주세요.
