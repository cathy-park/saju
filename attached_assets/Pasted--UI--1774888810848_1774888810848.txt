현재 사주 앱 "나의 흐름"의 홈 화면을 전체적으로 리디자인해주세요.

현재 상태는 기능 구조는 좋지만
UI가 아직 브랜드 앱처럼 보이지 않고
카드 스타일 / 버튼 스타일 / 컬러 사용이 일관되지 않아
전체적으로 촌스럽고 프로토타입 느낌이 납니다.

목표는

“사주 앱 느낌”
또는
“점집 느낌”

이 아니라

“토스처럼 정리된 한국형 모바일 서비스 UI + 감성 운세 앱”

입니다.

[1. 홈 화면 hero 캐릭터 수정]
--------------------------------

홈 화면 상단 캐릭터는 랜덤 캐릭터가 아니라

사용자의 일주 지지 동물 캐릭터

로 표시해주세요.

예:

일주 = 갑진 → dragon 캐릭터
일주 = 신묘 → rabbit 캐릭터
일주 = 병오 → horse 캐릭터

즉:

hero character = user's day pillar zodiac animal

로그인 전:

default 캐릭터 표시

로그인 후:

사용자의 일주 캐릭터 표시
