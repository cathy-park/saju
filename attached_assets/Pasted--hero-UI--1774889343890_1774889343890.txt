현재 사주 앱 "나의 흐름" 홈 화면을 다시 정리해주세요.

최근 수정 과정에서 hero 영역과 오늘의 에너지 영역이
점점 더 서비스 느낌이 아니라 임시 UI 느낌으로 변했습니다.

따라서 다음 기준으로 홈 화면을 다시 안정적인 구조로 정리해주세요.


--------------------------------
1. hero 영역 이전 구조로 복원
--------------------------------

현재 hero 영역을 이전 버전 스타일로 되돌려주세요.

구조:

날짜

캐릭터

닉네임님의 오늘 흐름

오늘의 흐름 핵심 해석 한 줄

오늘 일진 표시


그리고 hero 배경은

부드러운 연한 gradient 유지 가능하지만

카드처럼 보이지 않게

full width section 느낌으로 구성해주세요.


--------------------------------
2. hero 캐릭터 = 사용자 일주 동물 캐릭터
--------------------------------

홈 화면 캐릭터는 랜덤 캐릭터가 아니라

사용자의 일주 지지 동물 캐릭터

로 표시해주세요.

예:

갑진 → dragon

신묘 → rabbit

병오 → horse

즉:

hero character = user's day pillar zodiac animal


로그인 전:

default 캐릭터 표시


로그인 후:

user day pillar 기준 캐릭터 표시


--------------------------------
3. 로그인 전 사용자 이름 표시 수정
--------------------------------

로그인 전에도

"박소연님의 오늘 흐름"

으로 표시되고 있습니다.

이건 잘못된 상태입니다.


수정:

로그인 전:

"사용자님의 오늘 흐름"

또는

"오늘의 흐름"


로그인 후:

nickname 사용


nickname 없으면:

사용자 fallback


--------------------------------
4. 오늘의 에너지 영역 UI 변경
--------------------------------

현재 막대 그래프 3개 구조는

너무 단순하고 서비스 느낌이 약합니다.


다음 방식 중 하나로 변경해주세요:


옵션 A:

레이더 차트 형태

예:

재물운
관계운
건강운
학업운
일운


옵션 B:

물결 형태 multi-line 그래프


각 항목은

서로 다른 색상 사용


즉:

현재의 progress bar 3개 구조 제거


--------------------------------
5. 영역별 오늘의 운 카드 리디자인
--------------------------------

현재 카드가 설명 카드처럼 보입니다.


다음 구조로 변경해주세요:


카드 상단:

아이콘 + 영역명 + 상태 badge


카드 본문:

행동 추천 한 줄

결과 설명 한 줄


예:


관계운

먼저 연락해보세요

따뜻한 반응이 돌아올 가능성이 높아요


또한

카드 border 컬러 제거

상태 컬러는 badge에만 적용


--------------------------------
6. 카드 디자인 시스템 통일
--------------------------------

현재 카드 border 색이 서로 다릅니다.

다음 기준 적용:


card background = white

border = neutral gray 1px

shadow 제거

radius = 16


--------------------------------
7. CTA 버튼 구조 유지하지만 hierarchy 정리
--------------------------------

현재 구조는 좋습니다.


내 사주 보기

상대 추가


다음 방식으로 정리해주세요:


내 사주 보기

primary button

filled background


상대 추가

secondary button

outline button


--------------------------------
8. 섹션 타이틀 typography 정리
--------------------------------

다음 제목 스타일 통일:


오늘의 에너지

오늘의 흐름 구조

영역별 오늘의 운


font-size 18

font-weight 600

color #111


--------------------------------
9. 전체 UI 스타일 목표
--------------------------------

다음 기준 유지해주세요:


clean

soft

structured

Korean mobile service style

similar clarity level as Toss

fortune guidance companion app feeling

최종 목표
--------------------------------

서비스 UI 완성도는 아직 부족합니다.

목표는

감성 운세 앱이면서도
