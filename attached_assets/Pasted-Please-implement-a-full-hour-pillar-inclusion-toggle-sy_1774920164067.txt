Please implement a full hour-pillar inclusion toggle system in the saju app.

Currently users need to compare interpretations with and without the hour pillar (시주),
because birth time is often uncertain.

This feature must support:

1) include hour pillar
2) exclude hour pillar
3) compare both versions side-by-side

This applies to:
- saju interpretation
- five elements
- ten gods
- shinsal
- yongshin / secondary yongshin
- compatibility interpretation
- fortune flow interpretation (daewoon / sewoon / month / day)

- detailed ten gods
- shinsal
- yongshin

- interpretation summaries
